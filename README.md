# Ortodox Tro

En vacker introduktion till den ortodoxa kristna tron. Hitta ortodoxa kyrkor i Stockholm, vanliga frågor och resurser.

## Utveckling

```bash
# Installera beroenden
npm install

# Kör lokalt
npm run dev
```

## Funktioner

- **Karta**: Interaktiv karta över ortodoxa kyrkor i Stockholm.
- **FAQ**: Vanliga frågor om den ortodoxa tron.
- **Responsiv**: Fungerar på alla enheter.
